A. King of String Comparison

tag

  • 双指针

题意

两个串 s,ts, ts,t ，求 s[l,r]s[l, r]s[l,r] 字典序比 t[l,r]t[l, r]t[l,r] 小的数对 (l,r)(l, r)(l,r) 个数。
其中 1≤∣s∣=∣t∣,≤2⋅1051 \leq |s| = |t|, \leq 2 \cdot 10^51≤∣s∣=∣t∣,≤2⋅105 。

思路

枚举左端点，维护最长满足 s[l,r]=t[l,r]s[l, r] = t[l, r]s[l,r]=t[l,r] 的长度即可，长度有单调性，可以双指针维护。
复杂度 O(n)O(n)O(n) 。

其中 1≤k≤4,1≤n≤250000,1≤q≤20000,∣ai∣≤1081 \leq k \leq 4, 1 \leq n \leq 250000, 1 \leq q \leq 20000, |a_i| \leq 10^81≤k≤4,1≤n≤250000,1≤q≤20000,∣ai​∣≤108 。

C. Werewolves

tag

  • 树 dp

题意

nnn 个节点的树，求有多少个连通块使得其中所有点的点权存在绝对众数。
其中 1≤n≤30001 \leq n \leq 30001≤n≤3000 。

思路

枚举作为绝对众数的权值 colcolcol ，将权值等于 colcolcol 的点标记为 111 ，其余点标记为 −1-1−1 ，则问题转变为有多少个和大于 000 的连通块，跑树背包即可，所有情况下 111 的总数加起来为 nnn ，所以总复杂度为一般树背包的复杂度。
复杂度 O(n2)O(n^2)O(n2) 。

E. Replace Sort

tag

  • dp
  • jls 线段树

题意

给定长为 nnn 的序列 aaa 和大小为 mmm 的集合 bbb ，单次操作可以将 aaa 中若干个数字替换成 bbb 中的数字， bbb 中每个数字只能替换一次，求最小的操作次数使得 aaa 递增，无解输出 −1-1−1 。
其中 1≤n,m≤5⋅1051 \leq n, m \leq 5 \cdot 10^51≤n,m≤5⋅105 。

思路

先将 bbb 序列升序排序，容易有朴素的 dpdpdp ：

  • f[i]f[i]f[i] 表示 aaa 数组前 iii 个都递增，且最后一个数为 aia_iai​ 时的最小操作次数。
  • g[i][j]g[i][j]g[i][j] 表示 aaa 数组前 iii 个都递增，且最后一个数为 bjb_jbj​ 时的最小操作次数。
  • f[i]={f[i−1] (ai−1<ai)g[i−1][j] (bj<ai)f[i] = \begin{cases} f[i - 1]\ (a_{i - 1} < a_i)\\ g[i - 1][j]\ (b_j < a_i) \end{cases}f[i]={f[i−1] (ai−1​<ai​)g[i−1][j] (bj​<ai​)​
  • g[i][j]={g[i−1][j−1]+1f[i−1]+1 (ai−1<bj)g[i][j] = \begin{cases} g[i - 1][j - 1] + 1 \\ f[i - 1] + 1\ (a_{i - 1} < b_j) \end{cases}g[i][j]={g[i−1][j−1]+1f[i−1]+1 (ai−1​<bj​)​

复杂度为 O(n2)O(n^2)O(n2) 。
考虑优化， f[i]f[i]f[i] 的第一步转移可以直接更新，第二步转移相当于一个区间求 minminmin ， g[i]g[i]g[i] 的第一步转移相当于区间整体平移后加 111 ，第二步转移相当于区间取 minminmin 。
考虑用线段树优化转移，将 g[i]g[i]g[i] 的第二维表示在线段树上， f[i]f[i]f[i] 和 g[i]g[i]g[i] 的第二步转移相当于区间求 minminmin 和区间取 minminmin ，关键在于 g[i]g[i]g[i] 的第一步转移，可以考虑维护一个当前转移区间的起点 ststst ，区间平移时将 st−−st --st−− 即可，剩余的操作套 jlsjlsjls 线段树即可。
复杂度 O(n⋅logn)O(n \cdot logn)O(n⋅logn) 。

F. to Pay Respects

tag

  • 贪心

题意

一个人和恶龙对决，共 nnn 回合，每回合依次进行如下操作：

  • 恶龙吟唱治疗魔法，叠上一层治疗效果
  • 人吟唱伤害魔法，叠上一层伤害效果，若存在治疗效果则减少一层治疗效果
  • 人进行一次物理攻击
  • 所有魔法效果生效

人一次物理攻击造成 XXX 的伤害，假设第 iii 回合魔法生效前共有 ppp 层伤害效果和 rrr 层治疗效果，则第 iii 回合造成的伤害为 X+P⋅p−R⋅rX + P \cdot p - R \cdot rX+P⋅p−R⋅r ，人最多吟唱 KKK 次伤害魔法，给定长为 nnn 的 010101 串 sss ， si=1s_i = 1si​=1 表示龙第 iii 回合吟唱了治疗魔法。
最大化最后人能造成的伤害值。
其中 1≤n,X,P,R,≤106,0≤K≤N1 \leq n, X, P, R, \leq 10^6, 0 \leq K \leq N1≤n,X,P,R,≤106,0≤K≤N 。

思路

可以将治疗的伤害值等价为人造成的伤害值，初始时先将所有的治疗值加上。之后人在第 iii 回合吟唱魔法造成的总伤害可以等价地表示为 (n−i+1)⋅(P+[si=1]⋅r)(n - i + 1) \cdot (P + [s_i = 1] \cdot r)(n−i+1)⋅(P+[si​=1]⋅r) 。
计算出每回合的伤害值，取前 KKK 大的即可，物理攻击和治疗返回值单独计算。
复杂度 O(n⋅logn)O(n \cdot logn)O(n⋅logn) 。

G. Max Pair Matching

tag

  • 思维
  • 贪心

题意

n⋅2n \cdot 2n⋅2 个点的完全图，点 i,ji, ji,j 匹配的权值为 max(∣ai−aj∣,∣ai−bj∣,∣bi−aj∣,∣bi−bj∣)max(|a_i - a_j|, |a_i - b_j|, |b_i - a_j|, |b_i - b_j|)max(∣ai​−aj​∣,∣ai​−bj​∣,∣bi​−aj​∣,∣bi​−bj​∣) ，求图的最大权匹配。
其中 1≤n≤105,0≤ai,bi≤1091 \leq n \leq 10^5, 0 \leq a_i, b_i \leq 10^91≤n≤105,0≤ai​,bi​≤109 。

思路

点 i,ji, ji,j 的权值 max(∣ai−aj∣,∣ai−bj∣,∣bi−aj∣,∣bi−bj∣)max(|a_i - a_j|, |a_i - b_j|, |b_i - a_j|, |b_i - b_j|)max(∣ai​−aj​∣,∣ai​−bj​∣,∣bi​−aj​∣,∣bi​−bj​∣) 可以等价为 max(max(ai,bi)−min(aj,bj),max(aj,bj)−min(ai,bi))max(max(a_i, b_i) - min(a_j, b_j), max(a_j, b_j) - min(a_i, b_i))max(max(ai​,bi​)−min(aj​,bj​),max(aj​,bj​)−min(ai​,bi​)) 。所以原图中匹配的总权值可以等价地表示成选取 nnn 个点加上 max(a,b)max(a, b)max(a,b) ，剩下 nnn 个点减去 min(a,b)min(a, b)min(a,b) 。
假定 ai≥bia_i \geq b_iai​≥bi​ ，在数轴上比较一下可以发现 ∣ai−aj∣,∣bi−bj∣|a_i - a_j|, |b_i - b_j|∣ai​−aj​∣,∣bi​−bj​∣ 是没有作用的，假设 iii 与 jjj 匹配，取 max(ai,bi)max(a_i, b_i)max(ai​,bi​) 与 min(aj,bj)min(a_j, b_j)min(aj​,bj​) 的条件为 ai−bj≥aj−bia_i - b_j \geq a_j - b_iai​−bj​≥aj​−bi​ 或 bi−aj≥bj−aib_i - a_j \geq b_j - a_ibi​−aj​≥bj​−ai​ ，移项得 ai+bi≥aj+bja_i + b_i \geq a_j + b_jai​+bi​≥aj​+bj​ 。所以将点按 a+ba + ba+b 降序排序后取前 nnn 项加上 max(a,b)max(a, b)max(a,b) ，剩下的减去 min(a,b)min(a, b)min(a,b) 即为答案。
复杂度 O(n⋅logn)O(n \cdot logn)O(n⋅logn) 。

I. Flood Fill

tag

  • 网络流

题意

给定 n,mn, mn,m 的网格 aaa 和 bbb ，每一格为黑色或白色，单次操作可以选择 aaa 中一个同色连通块将其反色，求最后 a,ba, ba,b 中对应格子颜色不同的最小个数。
其中 1≤n,m≤1001 \leq n, m \leq 1001≤n,m≤100 。

思路

考虑所有同色的连通块，相邻的两个连通快不可能同时被反色，因为其中一个反色后就与另一个合并为同一个连通块了。
所以操作等价为：选取若干个互不相邻的连通块将其反色，求最后 a,ba, ba,b 中对应格子颜色不同的最小个数。
由于相邻的两个连通块颜色一定不同，所以所有连通块形成的图一定是二分图，即求一个二分图最大带权独立集，跑最大流即可。
复杂度 O(能过)O(能过)O(能过) 。

J. ABC Legacy

tag

  • 贪心

题意

给定长为 n⋅2n \cdot 2n⋅2 的只含 A,B,CA, B, CA,B,C 的串 sss ，判断能否将 sss 分割成若干个 AB,AC,BCAB, AC, BCAB,AC,BC 子序列，给出分割方式。
其中 1≤n≤1051 \leq n \leq 10^51≤n≤105 。

思路

将取子序列看成是括号匹配的过程， AAA 和 CCC 只能当作左括号和右括号，且确定了 A,CA, CA,C 的数量后 BBB 中有多少个需要作为左括号匹配，有多少个需要作为右括号匹配是确定的。假设有 xxx 个 BBB 需要作为左括号，将最靠左的 xxx 个 BBB 强制当成左括号，剩下的当成右括号匹配即可，可以证明这样一定最优，若不能全部匹配则无解。
复杂度 O(n)O(n)O(n) 。

K. Amazing Tree

tag

  • 贪心

题意

给定 nnn 个节点的无根树，编号为 [1,n][1, n][1,n] ，选定一个点为根，构造字典序最小的后序遍历序列。
其中 1≤n≤2⋅1051 \leq n \leq 2 \cdot 10^51≤n≤2⋅105 。

思路

想一下可以发现序列起点一定是某个度数为 111 的节点。
以编号最小的度数为 111 的节点为根，先预处理出每棵子树中叶子的最小权值，记为 minnminnminn 。
假设编号最小度数为 111 的节点为 ststst ，从 ststst 开始 dfsdfsdfs ，假设 dfsdfsdfs 中在不断地往父节点跳。每到达一个节点 xxx ，先检查所有子树的 minnminnminn 值，若 maxn(minn(v))(v∈son(x))maxn(minn(v))(v \in son(x))maxn(minn(v))(v∈son(x)) 小于 xxx ，则按 minn(v)minn(v)minn(v) 从小到大排序后直接后序遍历每棵子树 vvv ，最后再遍历 xxx 即可，若不是，则按 minn(v)minn(v)minn(v) 从小到大排序后先后序遍历除了 minn(v)minn(v)minn(v) 最大的子树 ，再遍历 xxx ，最后再继续从剩下的一个节点开始跳父亲。

L. Jason ABC

tag

  • 思维
  • 构造

题意

长为 n⋅3n \cdot 3n⋅3 的只含 A,B,CA, B, CA,B,C 的串 sss ，单次操作可以将 s[l,r]s[l, r]s[l,r] 全替换为 c∈{A,B,C}c \in \{A, B, C\}c∈{A,B,C} ，求最少操作次数使得 sss 中恰好有 nnn 个 A,B,CA, B, CA,B,C 。
其中 1≤n≤3⋅1051 \leq n \leq 3 \cdot 10^51≤n≤3⋅105 。

思路

若 sss 一开始就满足条件，则操作次数为 000 。
否则首先判断能否一次操作完成，即是否存在区间 [l,r][l, r][l,r] 使得 [1,l−1]∪[r+1,n⋅3][1, l - 1] \cup [r + 1, n \cdot 3][1,l−1]∪[r+1,n⋅3] 中存在两个字母满足条件，双指针查找即可。

tag

题意
