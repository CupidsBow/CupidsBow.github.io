abc266 Ex - Snuke Panic (2D)

发表于 2022-08-28 更新于 2024-07-21 分类于 杂题 本文字数： 3.8k 阅读时长 ≈ 3 分钟

树状数组优化 dp ，带权最长链

大体和官方题解差不多，只做个记录。

题意

二维坐标系上有 nnn 个关键点，你初始在 (0,0)(0, 0)(0,0) 位置，初始时刻为 000 ，你若在 tit_iti​ 时刻走到点 (xi,yi)(x_i, y_i)(xi​,yi​) 则会获得 aia_iai​ 的价值，最大化你能获得的价值。
其中你每一时刻可以进行下列两种操作之一：

  • 保持 xxx 坐标不变，或者将其变为 x−1x - 1x−1 或 x+1x + 1x+1 。
  • 保持 yyy 坐标不变，或者将其变为 y+1y + 1y+1 。

其中 1≤n≤105,1≤ti≤109,0≤xi,yi≤109,1≤ai≤1091 \leq n \leq 10^5, 1 \leq t_i \leq 10^9, 0 \leq x_i, y_i \leq 10^9, 1 \leq a_i \leq 10^91≤n≤105,1≤ti​≤109,0≤xi​,yi​≤109,1≤ai​≤109 。

思路

考虑朴素的 dpdpdp ，记 dp[t][x][y]dp[t][x][y]dp[t][x][y] 表示 ttt 时刻走到点 (x,y)(x, y)(x,y) 能获得的最大价值，则转移方程显然：

{dp[t][x][y]=max⁡(dp[t′][x′][y′])+val[x][y]t−t′≥∣x−x′∣+y−y′y≥y′\begin{cases} dp[t][x][y] = \max(dp[t'][x'][y']) + val[x][y]\\ t - t' \geq |x - x'| + y - y'\\ y \geq y'\\ \end{cases} ⎩⎪⎪⎨⎪⎪⎧​dp[t][x][y]=max(dp[t′][x′][y′])+val[x][y]t−t′≥∣x−x′∣+y−y′y≥y′​

转移合法的条件比较复杂，考虑对其进行转换以方便优化。

t−t′≥∣x−x′∣+y−y′  ⟺  {t−t′≥x−x′+y−y′t−t′≥−x+x′+y−y′  ⟺  {t−x−y≥t′−x′−y′t+x−y≥t′+x′−y′t - t' \geq |x - x'| + y - y'\\ \iff \begin{cases} t - t' \geq x - x' + y - y'\\ t - t' \geq - x + x' + y - y'\\ \end{cases}\\ \iff \begin{cases} t - x - y \geq t' - x' - y'\\ t + x - y \geq t' + x' - y'\\ \end{cases} t−t′≥∣x−x′∣+y−y′⟺{t−t′≥x−x′+y−y′t−t′≥−x+x′+y−y′​⟺{t−x−y≥t′−x′−y′t+x−y≥t′+x′−y′​

记 a=t−x−y,b=t+x−ya = t - x - y, b = t + x - ya=t−x−y,b=t+x−y ，并将原 dpdpdp 数组转变为 dp[y][a][b]dp[y][a][b]dp[y][a][b] ， y,a,by, a, by,a,b 的意义如上。
则有转移方程：

{dp[y][a][b]=max⁡(dp[y′][a′][b′])+val[x][y]y≥y′a≥a′b≥b′\begin{cases} dp[y][a][b] = \max(dp[y'][a'][b']) + val[x][y]\\ y \geq y'\\ a \geq a'\\ b \geq b'\\ \end{cases} ⎩⎪⎪⎪⎪⎨⎪⎪⎪⎪⎧​dp[y][a][b]=max(dp[y′][a′][b′])+val[x][y]y≥y′a≥a′b≥b′​

显然转移等价于求一个三维偏序带权最长链，并且我们只需要记录 nnn 个关键点对应的状态，对 yyy 排序后套二维树状数组 / 线段树优化 dpdpdp 或者套 cdqcdqcdq 分治即可，注意删去初始不能从 (0,0)(0, 0)(0,0) 到达的点，这部分点不能当作起点被计算。
总复杂度为 O(n⋅log2n)O(n \cdot log^2n)O(n⋅log2n) 。
