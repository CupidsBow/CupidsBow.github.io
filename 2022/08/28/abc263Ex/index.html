abc263 Ex - Intersection 2

发表于 2022-08-28 更新于 2024-07-21 分类于 杂题 本文字数： 2.3k 阅读时长 ≈ 2 分钟

二分 + 线段相交

题意

给定 nnn 条直线，求所有直线两两之间的交点（重点算多个）中离原点距离第 kkk 小的点的距离，直线表示为 A⋅x+B⋅y+C=0A \cdot x + B \cdot y + C = 0A⋅x+B⋅y+C=0 。
其中 1≤n≤5⋅104,−103≤∣Ai∣,∣Bi∣,∣Ci∣≤1031 \leq n \leq 5 \cdot 10^4, - 10^3 \leq |A_i|, |B_i|, |C_i| \leq 10^31≤n≤5⋅104,−103≤∣Ai​∣,∣Bi​∣,∣Ci​∣≤103 。

思路

注意到答案可以二分。
考虑二分答案，记二分值为 midmidmid ，作原点为圆心， midmidmid 为半径的圆，问题转换为求圆内的直线的交点个数。
每条直线与圆的两个交点可以转换为一个区间 [l,r][l, r][l,r] ，两直线交点在圆内等价于两直线代表的区间部分相交，问题转换为求区间对 [li,ri],[lj,rj][l_i, r_i], [l_j, r_j][li​,ri​],[lj​,rj​] 的个数，区间满足 li≤lj≤ri≤rjl_i \leq l_j \leq r_i \leq r_jli​≤lj​≤ri​≤rj​ ，按右端点排序后套树状数组求即可。
